# Cinema

Standalone Cinema download repo.

## Files

- `Cinema.apk` - the downloadable APK
- `index.html` - landing page for GitHub Pages
- `Start Cinema.cmd` - local launcher for Windows

## Publish

1. Push this folder to the `dude271/cinema` GitHub repository.
2. Enable GitHub Pages from the repository root.
3. Share the site URL and the direct APK URL.
